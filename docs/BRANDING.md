# Public name and icon directions

## Working name

**TetherPane** is the current public working name and repository slug. It connects the two central ideas—a phone tether that may be physical or wireless, and a focused Mac control pane—without borrowing an Android or Apple trademark.

An exact-name GitHub and general web search on 2026-07-14 found no current software product collision. That is useful screening, not legal trademark clearance. Recheck relevant trademark databases before a commercial 1.0 release.

## Other viable directions

- **Handset Relay** — clear and trustworthy, but “handset relay” already appears as generic telecommunications terminology.
- **BridgePane** — fits the Mac UI and multi-route connection model, but is less explicit about mirroring.
- **MirrorPort** — direct and memorable, but exact-name GitHub repositories already exist, so it is not recommended.

Avoid **AirDroid**, **MacDroid**, or names built around branded robot artwork. They overlap existing products or trademarks.
